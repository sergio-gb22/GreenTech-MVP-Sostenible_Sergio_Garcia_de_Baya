# GreenTech-MVP-Sostenible_Sergio_García_de_Baya

## Qué he hecho yo

He quitado todo lo innecesario que se podia sustituir por cosas que pesan menos, quedandome solo con HTML y CSS. Así la página no tiene que descargarse nada de fuera, el botón verde que había con Bootstrap lo he hecho yo con CSS normal y también le he puesto los efectos de :hover y :active que vimos en clase para que cuando pasas el ratón se note

## Lo que he usado

- **HTML semántico**: en vez de un <div> cualquiera he puesto <section> para la parte principal, y el <footer> va directo
- **CSS nativo**: todo lo que antes hacía Bootstrap ahora lo hago yo con CSS
- **Pseudoclases**: :hover y :active para el botón
- **Gradiente**: puse un linear-gradient negro encima de la foto para que el texto se vea mejor
